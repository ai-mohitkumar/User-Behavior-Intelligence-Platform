**Q: "Describe your clustering system"**

**Q: "What's unique about the ICSO metric?"**
A: "ICSO combines inter-cluster distance with intra-cluster variance: ICSO = mean_inter_cluster_distance / mean_intra_cluster_variance. Higher values indicate better cluster separation. This is research-publishable and not found in typical clustering projects."

**Q: "How does AutoML work?"**
A: "The system automatically tests KMeans, DBSCAN, and Hierarchical clustering algorithms on your data using a hybrid scoring function that combines Silhouette (50%), Davies-Bouldin (30%), and Calinski-Harabasz (20%) metrics. It then selects the optimal algorithm for your specific dataset characteristics."

**Q: "What about scalability?"**
A: "Tested on 148 records, suitable for real-time analysis up to ~50K records. For larger datasets, mini-batch KMeans and parallel processing can be implemented."
